# 📋 PROJECT FLOW

콘텐츠 제작 팀을 위한 프로젝트 스케줄 통합 관리 시스템

## 구조

```
GitHub Pages → Vercel (백엔드) → Google Sheets / Anthropic API
                   ↑
           작업자 / 관리자 (구글 로그인)
```

## 기능

- **관리자**: 프로젝트 생성·공정표·전체 작업 현황·작업자별 대시보드
- **작업자**: 내 작업 목록·캘린더·진행 상황 업데이트 (구글 계정 로그인만으로 사용)
- **AI**: 프로젝트 계획 자동 생성 (어떤 AI든 백엔드에서 교체 가능)
- **다중 사용자**: 외부 작업자도 링크 하나로 접근
